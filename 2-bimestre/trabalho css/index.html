A história do CSS


O que é CSS ?

introdução:

A história do CSS (Cascading Style Sheets) está ligada à evolução da própria web e à necessidade de separar conteúdo de apresentação.

Antes do CSS:

Nos primórdios da web, lá pelos anos 1990, páginas eram feitas apenas com HTML. Só que o HTML foi criado para estruturar conteúdo, não para definir aparência. Mesmo assim, desenvolvedores começaram a usar tags como font e atributos de cor para estilizar páginas, o que deixava o código bagunçado e difícil de manter.

A ideia do CSS

Em 1994, o cientista da computação Håkon Wium Lie propôs uma solução: um sistema separado para controlar o visual das páginas. A ideia era simples e poderosa — manter o conteúdo em HTML e o estilo em outro arquivo.

Impacto

O CSS transformou o desenvolvimento web:
Tornou o código mais organizado Facilitou manutenção e reutilização Permitiu designs muito mais sofisticados Hoje, praticamente toda página da web usa CSS, e ele continua evoluindo com novos recursos.
